# 🤖 AI DevOps Assistant

Пет-проект на стыке Artificial Intelligence и DevOps. 
Это веб-приложение, которое помогает разработчикам быстро генерировать базовую инфраструктуру (`Dockerfile` и `docker-compose.yml`) по текстовому описанию с помощью ИИ.

### 🛠 Стек технологий:
* **Упаковка:** Docker, Dockerfile
* **Оркестрация:** Docker Compose
* **CI/CD:** GitHub Actions (автоматическая сборка образа)
* **Приложение:** Python 3.10, Streamlit

### 🚀 Как запустить локально:
1. `git clone https://github.com/Syxarik2005/ai-devops-assistant.git`
2. `cd ai-devops-assistant`
3. `docker-compose up -d --build`
4. Откройте `http://localhost:8501` в браузере.
